# Bloco 2.5 - FastAPI e Pydantic

## Estrutura

- `exemplo-simples/`: Demonstração de API com validação Pydantic
- `exercicio/`: Exercício para praticar (com TODOs)
- `cheatsheet_fastapi.py`: Referência rápida
- `cheatsheet_pydantic.py`: Referência de validações

## Como Usar

1. Ver demonstração em `exemplo-simples/`
2. Fazer exercício em `exercicio/`
3. Consultar cheatsheets quando necessário

## Executar
```bash
# Exemplo
cd exemplo-simples
uvicorn main:app --reload

# Exercício
cd exercicio
uvicorn main:app --reload
```
